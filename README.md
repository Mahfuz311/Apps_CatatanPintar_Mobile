# 📱 Catatan Pintar

*Your AI-powered workspace*

Aplikasi *mobile* berbasis Android yang memudahkan pengguna untuk membuat dan mengelola catatan secara efisien. Aplikasi ini dilengkapi dengan fitur cerdas pemindaian teks (OCR) langsung dari kamera untuk mempercepat proses input data. 

## ✨ Fitur Utama

* **🔐 Autentikasi Pengguna:** Sistem Login dan Sign Up yang berjalan mulus menggunakan `SharedPreferences`.
* **👤 Profil Dinamis:** Halaman akun dengan sinkronisasi data *real-time* dan avatar inisial yang terbuat otomatis berdasarkan nama pengguna.
* **📷 Scan OCR (Optical Character Recognition):** Memindai dan mengekstrak teks dari dunia nyata melalui kamera pintar terintegrasi.
* **📝 Manajemen Catatan:** Menambah, menampilkan, dan mencari catatan dengan cepat menggunakan pencarian adaptif.
* **🛡️ Presentation Mode:** Keamanan UI yang dioptimalkan (Screen-Record Friendly) agar bebas dari blokir privasi saat melakukan demo aplikasi atau rekam layar.

## 🛠️ Teknologi yang Digunakan

* **Bahasa Pemrograman:** Java
* **IDE:** Android Studio
* **Database:** SQLite (`DatabaseHelper`)
* **Penyimpanan Sesi:** SharedPreferences (User Session Management)
* **Library Eksternal:** Google ML Kit Vision (Text Recognition)
